# Psychologist Booking & Media Platform

## Overview
This project is a web-based platform designed for a psychologist to manage client session bookings and publish educational media content. The system separates user and administrator (psychologist) roles and focuses on clarity, usability, and maintainable backend logic.

## Features
- Client-side interface for viewing information and requesting session bookings
- Secure authentication and role-based access
- Psychologist dashboard for managing bookings and publishing content
- Media section supporting podcasts, articles, and video materials
- Structured backend logic for content and session management

## Technologies Used
- Python
- Django & Django REST Framework
- HTML/CSS
- Git & GitHub

## Purpose
The project was developed to apply web development principles to a real-world use case, emphasizing user-oriented design, backend structure, and content management workflows.

## Future Improvements
- Email notifications for bookings
- Analytics on user interaction and content engagement
